# Computer Graphics - Learning & Practice

A comprehensive repository for learning and practicing computer graphics concepts, from basic shape drawing to 2D game development.

## 📋 Project Overview

This project contains hands-on examples and implementations covering various computer graphics topics:

- Basic shape drawing and transformations
- Animation and motion
- Color models
- 2D game development

## 📁 Project Structure


## 🚀 Getting Started

### Prerequisites

- **For Module 1:** Python 3.x with graphics library
- **For Module 2:** Modern web browser (Chrome, Firefox, Safari, Edge)

### Running Module 1 (Python Scripts)

```bash
python module-1/2-Drawing-Basic-Shapes.py
python module-1/5.circle-drawing.py
python module-1/6-Polygon-Filling.py
python module-1/9-scaling-transformtion.py
```

### Running Module 2 (Web-based)

Simply open the HTML files in your web browser:

```bash
# On Windows
start module-2/2-bouncing-ball-animation.html

# On macOS
open module-2/2-bouncing-ball-animation.html

# On Linux
xdg-open module-2/2-bouncing-ball-animation.html
```

Or drag and drop the HTML files into your browser.
